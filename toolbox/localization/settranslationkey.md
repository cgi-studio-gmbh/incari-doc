# Overview

![The Set Translation Key Node.](../../.gitbook/assets/settranslationkeynode.png)

The **Set Translation Key** sets the  translation `Key` for a specific **Object**.

[**Scope**](../overview.md#scopes): **Project**, **Scene**, **Function**, **Prefab**.

# Localization

## Introduction
 
 The **Localization Nodes** provide the functionality for **Objects** which utilize the `Translation` **Attribute**. In a *3D* **Scene**, these are [**Labels**](../../objects-and-types/scene-objects/3dobjects/label.md) and [**Texts**](../../objects-and-types/scene-objects/3dobjects/text.md). In **Scene2Ds**, these are [**Text 2Ds**](../../objects-and-types/scene2d-objects/figma/figmatext.md), [**Text Area 2Ds**](../../objects-and-types/scene2d-objects/gui/textarea.md), [**List 2Ds**](../../objects-and-types/scene2d-objects/gui/list.md), [**Dropdown 2Ds**](../../objects-and-types/scene2d-objects/gui/dropdown.md), and [**Button 2Ds**](../../objects-and-types/scene2d-objects/gui/button.md).

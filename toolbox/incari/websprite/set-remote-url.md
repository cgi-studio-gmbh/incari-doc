# Overview

![The Set Remote Url Node.](../../../.gitbook/assets/node-setremoteurl.png)

**Set Remote URL** sets the URL **Attribute** of a **Web Sprite** **Object**. It receives as inputs the **Object ID** of the **Web Sprite** **Object** and the URL.

# Overview

![The Set Material Property Node.]()

The **Set Material Property Node** sets a property, or properties, of a **Material**. These property are given their defining names in the **Attributes**. The names can be found by hovering over an **Attribute** of any of the **Shading Models** found in the [**Material Editor**](../../../modules/material-editor/README.md). 

Some examples of these property names are:

|Attribute|Property Name|

[**Scope**](../../overview.md#scopes): **Scene**, **Function**, **Prefab**

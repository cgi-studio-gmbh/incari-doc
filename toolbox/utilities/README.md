# Utilities

## Introduction

The _Utilities_ **Nodes** provide a means for a user to add additional functionality to an _Incari_ **Project**, whether that is creating a unique identifier; converting a `Data Type` to another one; grouping **Nodes** together; comparing an **Object** or `Data Type` with another one; changing or setting a **Color**; accessing an operating system's commands; or delaying or repeating a **Node**.
